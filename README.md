# 공개 프로젝트

### **Description**
 - 소스 코드 공개 목적으로 개발한 프로젝트.
 - 뱀게임에 TileMap으로 맵을 늘리고 루비 젠 등의 자체 기획을 담아서 개발한 2D 프로젝트.

### **Development Period**
 - 2024.03.18, 2024.03.20 (2일)

### **Development environment**
- Unity 2022.3.15f1
- Visual Studio 2019
- UGUI
- 2D
